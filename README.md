# Prismatix Public Starter

This repository is a clean-room, zero-history public frontend starter for teams that want a lightweight React shell in front of their own backend.

Included:

- Vite + React + TypeScript app shell
- Placeholder-only environment configuration
- Public-safe branding assets
- A minimal landing page for integrating your own backend

Designed for:

- Public starter releases
- Custom backend integrations
- Placeholder-only environment setup
- Simple frontend handoff and reuse

## Environment

Copy `.env.example` to `.env` and supply your own values:

```env
VITE_API_URL=https://YOUR_API_URL
VITE_PUBLIC_APP_NAME=Prismatix Public Starter
```

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
